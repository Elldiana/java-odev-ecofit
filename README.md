## Ecofit Uygulaması

Ecofit, kullanıcıların aktivitelerini takip edip kalori hesaplaması yapabildiği bir uygulamadır.  
Class yapıları kullanılmıştır. Veritabanı olarak **MySQL** kullanılmış ve kullanıcı kayıtları ile kalori bilgileri burada tutulmuştur.
